# Non-human experimental closure protocol

This is an exploratory continuity protocol fixed before B2 Class posteriors,
held-out B-1 clip 1002, or Material candidate margins are inspected. The
machine-readable source is `PROTOCOL.json`.

Class uses the already registered PANNs checkpoint, legacy coarse map,
event-restricted top-1 decision, cross-group margin `0.05`, confident-subset
agreement, `theta_commit=0.7`, and `A_independent>=0.9` video-determined
partition. Full 527-way and derived coarse posteriors are retained. The
multi-seed result is reported as curves, crossings/censoring, bootstrap
uncertainty, and variance components—not one first-crossing mean.

B-1 uses four deterministic calibration clips and keeps clip 1002 inaccessible
until the `q0.999` (NumPy `higher` convention) times two tolerance file is
written and hashed. Only equivalent same-forward operations gate lineage.
Successful lineage permits a fresh 816-trajectory recollection; failure is an
engineering status and ends internal probing.

Class probes are fixed across progress points, split only by video, selected in
inner grouped folds, and evaluated only through outer-fold predictions.
Fork-majority and deterministic-final targets remain separate.

Material is attempted only on clips whose existing UCS category is literally
an unambiguous material term. Captions and embeddings cannot create material
truth. Matching uses exact confident coarse Class, different UCS material,
scene fields, detector timing, and outcome-independent source-audio loudness.
If at least 20 videos and 640 cells cannot be matched before replay, the
workstream ends as `INCOMPLETE_ARTIFACTS`.

No rule in this file authorizes event-centered v2 claims, human-blocked axes,
B6, causal work, policy scheduling, confirmatory evaluation, a second
backbone, or paper writing.
